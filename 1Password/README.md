# 1Password 凭证管理与开发集成

本目录用于存放 TVBox 核心开发团队使用 [1Password](https://1password.com/) 进行安全协作的相关配置模板与指引。

## 为什么使用 1Password？
在 TVBox 的持续开发与测试中，我们会接触到各类接口 API、测试节点以及打包签名证书。为了确保这些敏感凭证不在代码仓库中明文泄露，我们采用 1Password 赞助的开源团队版进行统一管理。

## 开发者须知
1. 请勿在此目录以及整个项目中提交任何包含真实密码、Token 或私钥的明文文件。
2. 推荐使用 1Password CLI (`op`) 来将环境变量注入到本地测试中。
3. 相关配置模板可参考本目录下的其他示例文件。
