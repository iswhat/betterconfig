# 安全策略 (Security Policy)

## 凭证管理
为了保障开源项目的安全性，本项目开发和维护过程中涉及的所有测试账号、API 密钥以及其他敏感凭证，均使用 [1Password](https://1password.com/) 进行加密存储与安全管理。

感谢 1Password 为开源项目提供免费支持。

## 漏洞报告
如果您在 TVBox 中发现了安全漏洞，请通过 Issue 私密反馈，或直接联系维护者。

## 账号与凭证安全
如果你参与到本项目的核心开发并需要共享测试节点或接口账号，我们建议你使用安全的密码管理器。本项目的核心维护者使用 [1Password](https://1password.com/) 来管理相关开发凭证。
